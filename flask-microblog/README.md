# Flask-Microblog

### Introduction:

This is an example application based on Miguel Grinberg's book -- [Flask Mega-Tutorial](https://github.com/luhuisicnu/The-Flask-Mega-Tutorial-zh) .

Continuously updating.

```bash
> flask db init
> flask db migrate -m "First Migrate"
> flask db upgrade
> flask run
```

### Requirements:

- flask-login
- falsk-migrate
- flask-sqlalchemy
- flask-wtf
- python-dotenv
