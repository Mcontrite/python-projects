# Flask-Watchlist

### Introduction:

A simple movies watchlist based on Grey Li's flask tutorial -- [HelloFlask](https://read.helloflask.com/) .

```bash
> flask admin
> flask initdb
> flask forge
> flask run
```

### Requirements:

- Flask
- Flask-Login
- Flask-SQLAlchemy
- python-dotenv
- SQLAlchemy
