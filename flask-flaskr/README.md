# Flask-Tutorial

### Introduction:

A simple blog based on the Official Flask Document tutorial.

```bash
> flask init-db  # Initialized the database: instance/flaskr.sqlite

> set FLASK_APP=flaskr
> set FLASK_ENV=development
> flask run
```
