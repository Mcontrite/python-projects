# Flask-SayHello

### Introduction:

A simple message board called 'SayHello' based on Grey Li's flask tutorial -- [Github](https://github.com/greyli/sayhello) .

```
> flask initdb
> flask forge
> flask run
```

### Requirements:

- bootstrap-flask
- faker 
- flask-Moment
- flask-SQLAlchemy
- flask-WTF
- python-dotenv
